##  About The Project

**Tasty Bites Cafe** is a modern restaurant landing page designed using **Bootstrap 5**.
It demonstrates a clean UI, responsive layout, and essential restaurant website sections.

Perfect for:

* Restaurant landing pages
* Cafe websites
* Beginner frontend portfolio projects

---

##  Key Features

✔ Responsive design for all devices
✔ Sticky navigation bar
✔ Food menu cards
✔ Image gallery section
✔ Pricing / combo packages
✔ Contact form UI
✔ Google Maps integration
✔ Bootstrap modal popup
